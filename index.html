Open Source Library

Portable CAD in a small, self-contained runtime.

libcad is built for engineers who want CAD capability without heavyweight platform coupling. The goal is one compact core that can run in browser, native, and constrained environments.

Design goals

Portable first

Consistent behavior across runtimes, with minimal integration overhead.

Lightweight core

Small footprint for fast load times and easier distribution.

Self-contained

Fewer external dependencies and clearer operational boundaries.

For developers

Start from the docs to understand architecture and API surface, then validate behavior in the interactive demo.
